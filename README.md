# Sreeja Portfolio Website

## Description
This is a personal portfolio website created using HTML and CSS.
The website contains sections like About, Skills, Projects, and Contact.
It is fully responsive and designed as a beginner frontend project.

## Technologies Used
- HTML5
- CSS3

## Features
- Responsive Navigation Bar
- Hero Section
- About Section
- Skills Section
- Projects Section
- Contact Section
- Smooth Scrolling
- Hover Effects

## How to Run
1. Download the project files
2. Open the project folder
3. Double click on `index.html`

OR

Open `index.html` using any web browser.
